# echi-modal

> 开发Vue组件系列之模态框,主要有标题、内容、按钮个数、按钮颜色、按钮文案、按钮事件回调、遮罩层这些可配置项

### 使用命令行
```bash
$ Vue init webpack-simple echi-modal
$ Use sass? Yes
$ cd echi-modal
$ npm install
$ npm run dev
$ npm run build
```

### 项目结构
```bash
├── src                            # 项目源码。开发的时候代码写在这里。
│   ├── components                 # 组件目录
│   ├── App.vue                    # 项目根视图
│   ├── main.js                    # 程序主入口
├── index.html                     # 入口html文件
```
