# echi-modal

> 开发Vue组件系列之模态框,主要有标题、内容、定时器、按钮文案、按钮事件回调、遮罩层这些可配置项

项目基础工程文件是使用Vue-cli 3.0搭建的,这里不过多介绍

### 使用命令行
```bash
$ Vue create echi-modal
$ cd echi-modal
$ npm install
$ npm run serve
$ npm run build
$ npm run lint
```

### 项目结构
```bash
├── src                            # 项目源码。开发的时候代码写在这里。
│   ├── components                 # 组件目录
│   ├── App.vue                    # 项目根视图
│   ├── main.js                    # 程序主入口
```
